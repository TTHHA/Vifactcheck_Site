Key Features

Comprehensive Dataset

7,232 human-annotated claim-evidence pairs from reputable Vietnamese online news sources

  • 12 Diverse Topics
  • High-Quality Annotations
  • Multiple Evidence Sources

Advanced Models

State-of-the-art pre-trained and large language models for accurate fact-checking

  • Gemma Model Integration
  • Fine-tuning Capabilities
  • In-context Learning

High Performance

Superior accuracy with 89.90% macro F1 score on Vietnamese fact-checking

  • Robust Evaluation
  • Cross-domain Analysis
  • Error Analysis

Open Source

Complete toolkit including dataset, model checkpoints, and source code

  • GitHub Repository
  • Documentation
  • API Access
